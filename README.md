# TestGrizly
TestGrizly
При написании использовал Denwer для локального сервака
скачиваете денвер, запускаете его, в корневой директории веб интерфейса денвера создаёте любую папку и в неё закидываете весь проект 
для запуска необходимо скачать все файлы в одну папку, создать папку api, файл SUBMIT.php переместить в паппку api
если нужны изображения, то скачиваете архив assets, создаёте папку assets и разархивируете всё содержимое в эту папку 
далее в браузере переходите по пути "//localhost/"СОЗДАННАЯ ВАМИ ПАПКА"/main.php"
при отправке формы, данные сохраняются в файле json, находящийся в папке api, использовал как временное решение, поскольку в ТЗ не указано, какая именно бд должна быть
